# Pizza Sales Data Analysis & BI Metrics Repository

An end-to-end data analytics implementation focused on extracting transactional Key Performance Indicators (KPIs) and operational business insights from restaurant sales records using advanced SQL operations.

## 📈 Analytical Business Focus
This project structures raw operational records into actionable corporate intelligence, focusing on:
* **Core Performance KPIs:** Revenue calculations, average order value metrics, and consumer transaction volumes.
* **Temporal Trend Analysis:** Measuring sales fluctuations across specific hours, weekdays, and months to assist operational staffing models.
* **Market Share Segmentation:** Grouping performance metrics across menu categories and physical product sizes to evaluate sales distributions.
* **Menu Optimization Insights:** Tracking the highest and lowest grossing inventory items to flag revenue drags.

## 🛠️ Data Strategy & Architecture
* **Target Engine:** Microsoft SQL Server / MySQL Core
* **Techniques Demonstrated:** Aggregation functions (`SUM`, `COUNT`), dynamic type casting (`CAST` to `DECIMAL`), data part extractions (`DATENAME`, `DATEPART`), mathematical ratios, and subquery block structuring.
